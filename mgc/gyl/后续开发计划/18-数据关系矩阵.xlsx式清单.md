# 数据关系矩阵（Excel 式清单）

更新时间：2026-04-14  
适用范围：全系统核心表/集合  
说明：为便于复制到 Excel，本清单按“表/集合 -> 关联键 -> 上游/下游”方式整理。

## 1. 字段说明

| 列名 | 说明 |
| :--- | :--- |
| 表/集合 | 当前实体名称 |
| 关键键 | 当前实体最重要的主键/业务键 |
| 关联键 | 用于连接上下游的关键字段 |
| 上游来源 | 当前实体依赖的主对象 |
| 下游去向 | 当前实体影响的对象 |
| 关系类型 | 主事实 / 关系表 / 派生表 / 平台追踪 |
| 备注 | 补充说明 |

## 8. 最值得复制到 Excel 的五个优先核查对象

| 主题 | 核查主键 | 说明 |
| :--- | :--- | :--- |
| 订单统一 | `order_no` | 是否完整贯穿主单、明细、异常、分配、库存锁、履约、日志 |
| 库存统一 | `warehouse_code + sku_code + batch_no` | 是否以 `inventory_ledger` 为唯一事实层 |
| 经销授权统一 | `reseller_code + sku_code` | 是否以 `reseller_relation` 为主、`channel_dealer_authorizations` 为从 |
| 主数据统一 | `sku_code / warehouse_code / reseller_code` | 是否在主档、关系表、业务表、治理表中完全一致 |
| 流程统一 | `biz_type + biz_id` | 是否能稳定把业务对象映射到通知、待办、审批和消息 |
